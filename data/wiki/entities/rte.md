# RTE

Recognizing Textual Entailment. A benchmark for determining whether a target sentence is logically entailed by a source sentence.

## Overview

RTE is one of the earliest benchmarks for NLI, focusing on the binary task of entailment vs. non-entailment.

## Key Contributions / Capabilities

- **Logical Inference:** Tests the basic ability of a model to perform textual entailment.

## Connections

Mentioned in [[radford-2018-improving]] as a task where the first GPT model showed competitive but not always SOTA performance.
