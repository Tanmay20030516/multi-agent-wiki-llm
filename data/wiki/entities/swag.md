# SWAG

Sycophancy-free Word-Association Game (or similar). A benchmark for evaluating commonsense reasoning in language models.

## Overview

SWAG tests a model's ability to predict the most plausible continuation of a given scenario, requiring a basic understanding of how the world works.

## Key Contributions / Capabilities

- **Commonsense Reasoning:** Evaluates the model's ability to distinguish between plausible and implausible event sequences.

Used to evaluate [[bert]] in [[devlin-2019-bert]].
