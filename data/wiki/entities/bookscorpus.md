# BooksCorpus

A large corpus of unpublished books used for pre-training language models.

## Overview

BooksCorpus consists of thousands of books from various genres, providing a diverse set of long-form text that is useful for learning linguistic structures and long-range dependencies.

## Key Contributions / Capabilities

- **Diverse Pre-training Data:** Provided the scale and quality of text necessary for the first generative pre-training experiments.

## Connections

Used as the primary pre-training dataset for the original [[gpt]] model in [[radford-2018-improving]].
