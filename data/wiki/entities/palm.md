# PaLM

Pathways Language Model. A large-scale language model developed by Google.

## Overview

PaLM is a dense decoder-only Transformer model trained using the Pathways system, which allows for efficient training across thousands of TPU chips.

## Key Contributions / Capabilities

- **Scaling:** Demonstrated the benefits of scaling both model size and training data.
- **Reasoning:** Showed strong capabilities in chain-of-thought reasoning.
