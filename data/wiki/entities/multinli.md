# MultiNLI

Multi-Genre Natural Language Inference. A benchmark for evaluating a model's ability to determine if a premise entails, contradicts, or is neutral toward a hypothesis.

## Overview

MultiNLI is a large-scale dataset for NLI that spans multiple genres (e.g., fiction, government documents, transcripts), making it more challenging than single-genre datasets.

## Key Contributions / Capabilities

- **Textual Entailment:** Tests the model's ability to reason about the logical relationship between two sentences.

## Connections

Used to evaluate the first [[gpt]] model in [[radford-2018-improving]].
